CMDAnalysis 是一个终端历史命令可视化工具. 目前支持对bash, zsh 命令历史文件进行分析. 注重安全, 分析后即删除.使用下面命令来可视化历史文件, 每次调用都会生成一个url, 通过url可以访问可视化详细页面.

Star on Github

zsh

curl -F "file=@$HOME/.zsh_history" \
    http://127.0.0.1:8000/api/history/

bash

curl -F "file=@$HOME/.bash_history" \
    http://127.0.0.1:8000/api/history/
